# Frontend ligero - Juego Estrategia

Cliente minimal en HTML + Vanilla JS que consume la API del backend en `http://localhost:3000`.

Instrucciones rápidas:

1. Inicia el backend:

```bash
cd backend
npm install
npm run dev
```

2. Sirve los archivos estáticos:

```bash
cd frontend
python3 -m http.server 5173
# Abrir http://localhost:5173
```
